## kubernetes: kubectl apply -f k8s/ --recursive

## Symulator serwera na platformie discord. Projekt na BAW i Technologie aplikacji Chmurowych.

## Dla dockera:
- ip na ip urzadzenia
- skomentowac volumen mongo i odkomentowac zakomentowany volumen
- dodac do keycloak environment:
  <code>
    environment:
      - KC_DB=dev-file
      - KC_DB_USERNAME=sa
      - KC_DB_PASSWORD=
  </code>
